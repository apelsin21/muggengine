muggengine
==========

Game Engine written in C/C++ primarily meant to be used through Lua.

Dependencies:

    SDL2
    
    ENet
    
    Bullet
    
    Assimp
    
    GLEW
    
    FreeType2
    
    FreeImage
    
    GLM

Requirements:

    Platform: Whatever x86 platforms SDL2 runs on.
    
    GPU: OpenGL 3.0+, GLSL 1.30+ (OpenGL ES isn't supported)
